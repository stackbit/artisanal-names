# artisanal-names

A name generator using hand-picked nouns and adjectives.

## Installation

```shell
npm install @stackbit/artisanal-names
```

## Usage

```javascript
const nameGenerator = require('@stackbit/artisanal-names');
nameGenerator.generate();
```

## Examples

```shell
> nameGenerator.generate()
'cool-panda'

> nameGenerator.generate()
'majestic-bee'

> nameGenerator.generate()
'enthusiastic-tomato'
```
